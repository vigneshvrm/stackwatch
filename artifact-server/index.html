How It Works

StackWatch acts as a secure reverse proxy, unifying all your monitoring tools

Your Browser
→
StackWatch Gateway
→
Prometheus / Grafana

Instead of remembering multiple URLs and ports (Prometheus:9090, Grafana:3000, etc.), you access everything through one clean interface. Internal ports are blocked by firewall rules, ensuring all traffic flows through the secure StackWatch gateway.

Installation Guide

Get StackWatch up and running in minutes

1

One-Command Install (Recommended)

The fastest way to install StackWatch - downloads, extracts, and deploys automatically:

curl -fsSL https://artifact.stackwatch.io/install.sh | sudo bash
2

Access StackWatch

Once installation completes, open your browser and navigate to your server's IP address:

# Check your server IP
hostname -I | awk '{print $1}'

You'll see the StackWatch dashboard with access to Prometheus, Grafana, and system health.

3

Verify Installation

Run the health check to ensure all services are running:

sudo /opt/stackwatch/scripts/health-check.sh

Manual Installation

Prefer to install manually? Follow these steps

1

Download the Package

Download the latest release from the download section above or use curl:

# Download latest stable version
curl -LO https://artifact.stackwatch.io/latest/stackwatch-latest.tar.gz
2

Extract to /opt/stackwatch

Extract the package to the installation directory:

sudo mkdir -p /opt
sudo tar -xzf stackwatch-latest.tar.gz -C /opt
sudo mv /opt/stackwatch-* /opt/stackwatch
3

Run the Deployment Script

Execute the deployment script to configure and start all services:

sudo /opt/stackwatch/scripts/deploy-from-opt.sh
