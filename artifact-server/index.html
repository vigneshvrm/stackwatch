📦 Installation Guide

  1. Download the package

    Click the download button above or use curl:

    # Download latest stable version
    curl -LO https://artifact.stackwatch.io/stackwatch/build/2026/01/latest/stackwatch-latest.tar.gz

  2. Extract the package

    Unpack the downloaded archive:

    tar -xzf stackwatch-latest.tar.gz
    cd stackwatch

  3. Run the installer

    Execute the deployment script with sudo:

    sudo ./deploy-stackwatch.sh

  4. Access StackWatch

    Open your browser and navigate to your server's domain or IP address.
